# Hookshot Transformations

JavaScript transformation scripts for [Matrix Hookshot](https://matrix-org.github.io/matrix-hookshot/) to parse webhook payloads and produce Matrix messages.

## Available Scripts

| Script | App |
|--------|-----|
| `bazarr.js` | [Bazarr](https://www.bazarr.media) |
| `dispatcharr.js` | [Dispatcharr](https://github.com/dispatcharr/dispatcharr) |
| `epic-clabs.js` | [epicgames-freegames-node](https://github.com/claabs/epicgames-freegames-node) |
| `grafana.js` | [Grafana](https://grafana.com) |
| `jellyfin.js` | [Jellyfin](https://jellyfin.org) |
| `mastodon.js` | [Mastodon](https://joinmastodon.org) |

| `proxmox.js` | [Proxmox](https://www.proxmox.com) |
| `radarr.js` | [Radarr](https://radarr.video) |
| `sonarr.js` | [Sonarr](https://sonarr.tv) |
